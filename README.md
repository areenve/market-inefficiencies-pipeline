# Market Inefficiencies Project 

Notebook runner for a market microstructure / inefficiencies pipeline (cross-venue price dislocations).

## What it does
- Upload a project bundle (zip) to Colab
- Install dependencies
- Run collection → dislocation detection → backtest

## How to run
Open `notebooks/projcode.ipynb` in Google Colab and run cells top to bottom.

## Notes
Large datasets, outputs, and logs are excluded via `.gitignore`.
